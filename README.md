# 🚀 Ejercicios Prácticos de C# y .NET

¡Bienvenido a este repositorio de aprendizaje! Aquí encontrarás una colección organizada de ejercicios prácticos en **C#** y el ecosistema **.NET**, diseñados para dominar desde los conceptos fundamentales de la Programación Orientada a Objetos hasta arquitecturas y patrones avanzados.

El repositorio contiene **3 ejercicios prácticos por cada tema**, incrementando su nivel de complejidad.

---
